# MoveisCasa

### Sobre o Projeto
O **MoveisCasa** é uma landing page desenvolvida para uma loja de móveis, focada em apresentar produtos de forma elegante e organizada. O projeto prioriza a experiência do usuário com um design responsivo e uma interface intuitiva para navegação no catálogo.

### Tecnologias Utilizadas
* **HTML5**: Estruturação semântica do conteúdo.
* **Tailwind CSS**: Estilização moderna e layout responsivo.
* **JavaScript**: Interatividade e dinamismo nas seções.
